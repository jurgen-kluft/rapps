# Magic Cube

This is a cube that contains:

- 3D printed enclosure (75 mm x 75 mm x 75 mm)
- Firebeetle 2, ESP32 microcontroller
- SC7A20H, a 3-axis accelerometer (ultra low power)
- Battery (900 mAh)

## Functionality

- Detects orientation and movement
- Can determine current up face of the cube
- Can detect single and double taps
- Sends 'event' data over WiFi using UDP
